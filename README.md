# Приложение для детекции ядер и анализа ROI (PyQt5)

Десктопное приложение для анализа гистологических изображений:
- детекция ядер нейросетью;
- калибровка масштаба (px/mm);
- разметка ROI (прямоугольник/полигон);
- расчёт площади, количества ядер и плотности;
- одиночная и пакетная обработка;
- экспорт результатов в CSV/Excel.

## Структура проекта
- `main.py` — точка входа и запуск GUI.
- `ui.py` — интерфейс PyQt5 и пользовательские сценарии.
- `backend.py` — загрузка изображений, детекция, расчёты, экспорт.
- `requirements.txt` — зависимости.

## Поддерживаемые форматы
- TIFF: `.tif`, `.tiff`
- PNG: `.png`
- JPEG: `.jpg`, `.jpeg`

## Установка

### 1) Создайте и активируйте виртуальное окружение
Linux/macOS:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

Windows (PowerShell):
```powershell
py -m venv .venv
.venv\Scripts\Activate.ps1
```

### 2) Установите зависимости
```bash
pip install -U pip
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```

## Быстрый сценарий работы
1. При старте приложение **обязательно** запросит калибровку масштаба.
2. Нажмите `Калибровать масштаб`:
   - откройте фото с линейкой;
   - проведите линию по известной длине;
   - введите длину в мм.
3. Нажмите `Открыть изображение` и загрузите снимок.
4. При необходимости настройте цветокоррекцию (цветность/яркость/контрастность/резкость).
5. (Рекомендуется) Нажмите `Подобрать по клетке` и выделите среднее ядро.
6. Нажмите `Детектировать ядра`.
7. Разметьте ROI инструментами `Прямоугольник` или `Полигон`.
8. Смотрите таблицу справа: площадь (мм²), ядра, плотность.
9. Нажмите `Экспортировать` для сохранения отчёта.

## Детекторы
В `Параметры детекции` можно выбрать встроенный детектор:
- `Cellpose Nuclei (рекомендуется для CPU)` — основной вариант по умолчанию.
- `StarDist (классический вариант)`.

Также можно загрузить пользовательскую модель кнопкой:
- `Загрузить модель (.pt/.onnx)`

Если загружена пользовательская модель, используется она.

## Пакетная обработка
Кнопка: `Пакетная обработка`

Сценарий:
1. Выберите папку с изображениями.
2. Выберите режим:
   - `По всему кадру`;
   - `По прямоугольникам`;
   - `По полигонам`.
3. На первом изображении задайте параметры цветокоррекции.
4. При необходимости разметьте ROI для каждого изображения.
5. Запустите обработку и сохраните единый отчёт (`CSV`/`Excel`).

## Горячие клавиши
- `Ctrl+Z` — отмена последнего действия.
- `Delete` — удалить выбранные ROI.
- `Esc` — отменить текущее рисование.

## Типичные проблемы

### 1) Ошибка Qt plugin `xcb` на Linux
Если при запуске есть сообщение про `Could not load the Qt platform plugin "xcb"`, установите системный пакет:
```bash
sudo apt-get install libxcb-xinerama0
```

### 2) Сообщение `QStandardPaths: wrong permissions on runtime directory`
Обычно это предупреждение окружения Linux. На работу приложения чаще всего не влияет.

### 3) Сообщение, что встроенная нейросеть недоступна
Проверьте зависимости из `requirements.txt` и переустановите:
```bash
pip install -r requirements.txt
```

### 4) Проверка установки нейросетей
```bash
python -c "import cellpose, stardist; print('ok')"
```

## Примечания
- Детекция выполняется в отдельном процессе, чтобы UI не зависал.
- Масштаб обязателен: без него расчёты в мм² и плотность недоступны.
