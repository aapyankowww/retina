# Анализ клеточных ядер (PyQt5 + StarDist)

Десктопное приложение для анализа гистологических снимков (H&E):
- детекция ядер нейросетью StarDist (`2D_versatile_he`);
- калибровка пространственного масштаба (px/мм);
- разметка областей интереса (ROI): прямоугольник, полигон;
- расчёт площади, количества ядер и плотности по каждому ROI;
- одиночная и пакетная обработка папок;
- экспорт результатов в CSV/Excel.

## Структура проекта
| Файл | Назначение |
|---|---|
| `main.py` | Точка входа, запуск GUI |
| `ui.py` | Интерфейс PyQt5 |
| `backend.py` | Детекция, обработка изображений, экспорт |
| `requirements.txt` | Зависимости Python |
| `Retina.cmd` | Быстрый запуск на Windows (двойной клик) |

## Поддерживаемые форматы изображений
`.tif` · `.tiff` · `.png` · `.jpg` · `.jpeg`

## Установка

### 1. Создайте виртуальное окружение

**Windows (PowerShell):**
```powershell
py -m venv .venv
.venv\Scripts\Activate.ps1
```

**Linux / macOS:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 2. Установите зависимости
```bash
pip install -U pip
pip install -r requirements.txt
```

> Первый запуск может занять время — StarDist скачает веса модели (~20 МБ).

## Запуск

**Windows:** дважды кликните `Retina.cmd`
**Из терминала:**
```bash
python main.py
```

## Сценарий работы

1. **(Опционально) Шаг 1 — Масштаб.** Нажмите «Калибровать масштаб», откройте фото линейки, проведите линию, введите реальную длину. Без этого площадь и плотность будут недоступны, детекция работает.
2. **Шаг 2 — Изображение.** Нажмите «Открыть изображение» и загрузите снимок.
3. **Шаг 3 — Цветокоррекция.** При необходимости подправьте насыщенность/яркость/контраст/резкость.
4. **Шаг 4 — Детекция.** Нажмите «Запустить детекцию». Ядра будут отмечены зелёным контуром.
5. **Разметка ROI.** Используйте кнопки «Прямоугольник» / «Полигон» на панели над изображением.
6. **Шаг 5 — Экспорт.** Нажмите «Экспортировать» в правой панели для сохранения отчёта.

## Параметры нейросети

Кнопка «Параметры нейросети» открывает расширенные настройки:
- порог уверенности и порог разделения соседних ядер;
- upscale-фактор (по умолчанию 2.05 — оптимально для H&E);
- нормализация цвета Reinhard;
- фильтр по окраске гематоксилином.

Пресеты: **Точный** (меньше ложных срабатываний) и **Чувствительный** (больше ядер).

## Пакетная обработка

1. «Дополнительные настройки» → «Пакетная обработка».
2. Выберите папку с изображениями и режим (`По всему кадру` / `По прямоугольникам` / `По полигонам`).
3. Настройте цветокоррекцию на первом снимке.
4. При необходимости разметьте ROI для каждого файла.
5. Сохраните сводный отчёт (CSV / Excel).

## Горячие клавиши
| Клавиша | Действие |
|---|---|
| `Ctrl+Z` | Отменить последнее действие |
| `Delete` | Удалить выбранные ROI |
| `Esc` | Отменить текущее рисование |

## Типичные проблемы

**Qt plugin `xcb` на Linux:**
```bash
sudo apt-get install libxcb-xinerama0
```

**Нейросеть недоступна — «Встроенная модель недоступна»:**
```bash
pip install -r requirements.txt
python -c "import stardist; print('stardist ok')"
```

**Excel-экспорт не работает:**
```bash
pip install openpyxl
```

## Примечания
- Детекция выполняется в отдельном процессе — UI не зависает.
- Без калибровки масштаба расчёты в мм² и плотность показываются как «—».
- Первая детекция дольше: StarDist компилирует граф TensorFlow.
